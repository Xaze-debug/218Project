# 218Projet1
1. Скачивание
Приложение доступно для большинства современных операционных систем. 
Официальный сайт: Перейдите на obsidian.md/download.
Windows: Нажмите Download for Windows(рекомендуется версия Installer .exe).
macOS: Скачайте файл Universal и перетащите Obsidian в папку Applications.
Мобильные устройства: Приложение доступно в App Store (iOS) и Google Play(Android).
Linux: Доступны пакеты .AppImage, .deb, .snap и Flatpak. 

2. Установка и запуск
Запустите скачанный файл и следуйте подсказкам на экране для завершения установки.
При первом запуске вам будет предложено создать «Хранилище» (Vault). Это обычная папка на вашем компьютере, где будут храниться все заметки в формате Markdown.
Выберите Create new vault, укажите название и путь к папке. 

3. Активация и лицензия
Obsidian не требует обязательной «активации» в привычном смысле для работы базовых функций. 
Личное использование: Программа полностью бесплатна. Создание аккаунта для этого не требуется.
Коммерческое использование: Если вы используете Obsidian в компании с двумя и более сотрудниками, необходимо приобрести коммерческую лицензию на официальном сайте.
Аккаунт: Регистрация (Sign in → Create account на сайте) нужна только для платных сервисов, таких как Obsidian Sync(официальная синхронизация) или Obsidian Publish. 

4. Настройка (рекомендации 2026)
Язык: Зайдите в Settings (шестеренка) → Appearance → Language, чтобы выбрать русский язык.
Плагины: Для расширения функций перейдите в Settings → Community plugins, нажмите Turn on community plugins и используйте кнопку Browse для поиска нужных инструментов.
Синхронизация: Если вы не используете платный Obsidian Sync, можно бесплатно синхронизировать заметки через Google Drive, iCloud, Dropbox или сторонние решения вроде Syncthing.
